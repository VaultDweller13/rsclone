# Webpack-config

Webpack 5 configured to work with:

- TypeScript
- SASS/SCSS
- Jest
- ESLint
  - airbnb-base,
  - airbnb-typescript/base,
  - eslint:recommended
  - @typescript-eslint/recommended
  - @typescript-eslint/recommended-requiring-type-checking"
- Prettier

# Usage

Start dev-server:  
`npm run start`

Build:  
`npm run build`

Test:  
`npm run test`
